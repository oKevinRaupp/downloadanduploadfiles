# Download and Upload

API para fazer download e upload de arquivos

# File:

<b>GET</b>

api/v1/file/download/{filename} -> Faz download do arquivo

<b>POST</b>

api/v1/file/upload -> Envia um arquivo

api/v1/file/uploadMultiple -> Envia varios arquivos
